OCL.js – A simple implementation of Object Constraint Language

The OCL ist a language used in Meta-Modelling to add additional constraints to the meta model which can't be expressed by the given meta modelling language. Since there do exist implementations in Java (e.g. Eclipse Modelling Framework EMF), this is a basic OCL implementation in JavaScript.

This page contains some example OCL constraints which are evaluated live while this page loads. To create a use case, the following images shows two connected classes which describe the following: A person can own multiple cars. A car belongs to just one person. A car has a color. A person has a name as well as its age.
